Frontend:
1. Сайт должен быть доступен на всех типах устройств - широкие экраны, десктоп, лептоп,мобильные девайсы
2. Кроссбраузерная верстка
3. Семантически валидная верстка
4. Сайт должен быстро грузится - оценка от Гугл на уровне 99%-100%
5. Верстка пиксель-перфект.
6. Интерактивные элементы должны быть анимированы.
7. Минимум js.
8. Без использования CSS фреймворков.
9. Обязательное использование сборщика.

CMS: 
1. Возможность через админ-панель редактировать оба меню и его елементы
2. Возможность через админ-панель редактировать страницы (без использования HTML разметки)
3. Возможность через админ-панель создавать однотипные страницы.
4. Возможность повторно использовать блоки контента
5. При подтверждении формы - полученные данные должны быть доступны для просмотра в админ панели
6. При подтверждении формы - на почту админа должно приходить уведомление о новой заявке
